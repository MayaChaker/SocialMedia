# SocialMedia

A simple social media UI built with React. It shows a typical layout with:

- Top navigation bar with search, notifications, and user profile
- Left sidebar with navigation links and a friends list
- Central feed with stories, a share box, and posts
- Right sidebar with birthdays, an ad card, and online friends

This project was created with Create React App and styled using plain CSS.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
